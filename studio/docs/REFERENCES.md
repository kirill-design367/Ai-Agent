### Синтез для AUREA (что берём, как переводим в концепцию)
- **Экстремальный контраст кеглей** (микро ↔ дисплей) — есть у всех 4. → √φ-шкала.
- **Тёплый монохром** — тёмная тёплая база (ближе к sondaven по регистру, но гротеск).
- **ОДИН акцент** — у всех цветной; у нас **тёплый свет без hue** (сложнее — держим
  «дорого» глубиной/светом, а не краской; это и есть отличие и риск, следим).
- **Нумерованные редакционные секции** — Martian Mono нумерация/лейблы.
- **Структура студии** (digitalists): hero-манифест → индекс работ (реф-номера) →
  услуги-список → большой CTA → тёмный тип-футер.
- **Живая деталь-ремесло** (часы у sakazuki) — добавим ненавязчиво (напр. МСК-время).
- **Наше преимущество:** ни у кого нет ОДНОЙ идеи-жеста. «Разворот из точки» —
  наш ownable-крючок, которого у них нет.

## Общие принципы «дорогого» (уверенно, от просмотра не зависит)

1. **Контраст масштабов + воздух важнее украшений.** Дорогие сайты строят иерархию
   огромной разницей кеглей (микро-лейбл ↔ дисплей) и большими пустыми полями, а не
   декором. → У нас: √φ-шкала (§3) + 60–70% воздуха.
2. **Мало движения, но точное.** 2–3 момента-манифеста, остальное спокойно; motion
   мотивирован, с одной кривой. → Наша «Иерархия интенсивности» (§1.3).
3. **Один материал.** Единая сетка, единый ритм, один тип перехода, одна кривая —
   ощущение «одной руки». → theme/motion.ts + theme/tokens.
4. **Переходы вместо перезагрузок.** Постоянный shell, контент уходит/приходит
   осмысленно (не фейд). → View Transitions + GSAP, «сворачивается к точке».
5. **Детали доведены до конца.** Курсор, `::selection`, скроллбар, фокус, 404,
   пустые состояния — ничего дефолтного.

## Анти-паттерны (что делает сайт дешёвым — от просмотра НЕ зависит)
- **Золото на тёмном** как сигнал «премиум» (§1.1) — первый маркер дешёвого РФ-сайта.
- **Дефолтный `ease-in-out`** и одинаковая длительность на всё — «шаблонный» motion.
- **Сток-иконки, эмодзи, дефолтные тени** (`box-shadow: 0 4px 8px rgba(0,0,0,.1)`).
- **Одинаковые карточки-прямоугольники** в равномерной сетке без иерархии.
- **Центрирование всего подряд** — нет напряжения композиции (нам нужна φ-асимметрия).
- **Градиентные кнопки**, «стеклянные» карточки без причины, радуга-градиенты.
- **Мелкий контраст кеглей** (H1 лишь чуть крупнее body) — «корпоративная гладкость».
- **Непрерывное движение** (всё всегда шевелится) — утомляет, читается как дешёвое.
- **Буллеты-кружочки, точки-россыпи** как декор (§1.2).
- **Латинский «премиум»-шрифт с fallback-кириллицей** — кириллица должна быть родной.
