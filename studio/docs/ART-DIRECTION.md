# AUREA — Арт-дирекшен. «От точки до шедевра»

> Единая концепция сайта. Из неё выводятся ВСЕ решения: типографика, motion,
> композиция, свет, переходы, детали. Если элемент не выводится отсюда — его нет.

## Концепция (ядро)

1. AUREA — про **превращение**: из одной точки (сырая идея, пустой холст,
   «нам нужен сайт») рождается завершённая форма — шедевр.
2. Сайт не *описывает* эту услугу — он её **показывает**: каждый экран есть акт
   организации материи из минимума в структуру, и посетитель видит это
   превращение в реальном времени. Сайт — доказательство, а не рассказ.
3. **Атом бренда — точка.** Она живёт на самом мелком масштабе (курсор, буллет,
   точка в конце строки, засечка) и по мере скролла множится и выравнивается:
   точки → линии → сетки → буквы → изображения → целая композиция.
4. Материал — **свет на почти-чёрном**. `#0a0a0b` — это глубокое пространство,
   «отсутствие света», из которого форма *проявляется*, а не заливка-фон.
5. **Пустота — сцена, а не фон.** Дорогое всегда окружено воздухом; шедевр видно
   только там, где вокруг ничего не спорит. Плотность чередуется с паузой, как
   в музыке.

Имя работает дважды: **aurum** (золото — акцент-свет) и **aurea sectio**
(золотое сечение — пропорция композиции). Оба смысла — в системе ниже.

## Принципы (система, из которой выводим решения)

### Motion — «сборка из точки», не «появление»
- Элементы **собираются**, а не фейдятся: точки слетаются в линию, линия
  вычерчивается, буква проявляется штрихом. Метафора одна — рост/кристаллизация.
- **Одна семья кривых** («AUREA ease»): медленный старт из точки → уверенное
  ускорение → мягкая посадка. `ease-out` доминирует — материя *оседает* в форму.
  Никаких дефолтных `ease-in-out`, ничего не мигает и не «выезжает» механически.
- Иерархия входа: сначала — смысловой якорь (то, что ведёт взгляд), затем
  структура вокруг него, затем детали. Stagger — ритмичный, из одной точки-истока.

### Композиция — золотое сечение и один герой
- Строится по **φ (1:1.618)**: смысловой якорь смещён по золотому сечению, не по
  центру; асимметрия управляемая. Один герой на экран.
- **Контраст масштабов как главный инструмент**: микро-лейблы (10–12px, uppercase,
  трекинг +0.2em) против дисплейных кеглей. Русский — работаем с длинными словами,
  переносами, кернингом; заголовок — объект, а не строка.
- **Сетка видима как след**: тонкие направляющие проявляются при скролле — «леса»,
  по которым росла форма, потом уходят.

### Пустота — активный материал
- ~60–70% экрана — воздух. Роль: дать шедевру дышать и держать ритм
  **сжатие → расширение** (плотный блок → огромная пауза → одна фраза в пустоте).

### Переходы между страницами — непрерывность, не перезагрузка
- Постоянный **shell**; при переходе текущая страница «сворачивается к точке»
  (или к общему элементу — заголовку/точке), новая «вырастает из точки».
- View Transitions API + свой GSAP-слой. ~600–800 ms. Переход — часть композиции:
  рассказывает «свернулось → развернулось», сохраняя ощущение одного пространства.

### Глубина — z-пространство, свет, фактура
- Слои с параллаксом; **grain/noise** поверх; один источник света (световая точка)
  задаёт объём и градиент. Плоскости нет — есть ближе/дальше и свет на гранях.

### Цвет и свет — люкс-монохром + дозированное золото
- Монохром на `#0a0a0b`; акцент — **тёплое золото (aurum)**, но дозой: не «золотой
  сайт», а золото как свет на краях (подчёркивание, точка, `::selection`, кромка
  карточки на hover). Один рычаг — `--accent`.

### Детали, по которым узнаётся дорогая работа
- **Курсор — та самая точка бренда**, ставшая объектом: кружок, который реагирует
  (растёт на ссылках, становится «смотреть» на кейсах, магнитится к кнопкам).
- `::selection` золотой; подчёркивания **вычерчиваются**; скроллбар тонкий;
  фокус-кольца фирменные; состояния формы и пустые состояния — оформлены;
  **404 — «точка потерялась»**. Ничего дефолтного не остаётся.

### Загрузка (главная) — бренд-момент
- Точка в центре пустого чёрного за ~1.2 s **прорастает** в вордмарк / первую
  композицию. Первый показ метафоры, а не заставка ради заставки.

### Reduce-motion — богато, но тихо
- НЕ упрощённая версия. Та же вёрстка, глубина и типографика; движение
  **приглушается** (амплитуда → к нулю, кристаллизация становится мягким
  проявлением), но композиция и премиальность сохраняются.

## Как концепция прослеживается по типам страниц
- **Главная** — манифест: полное превращение точка → композиция студии.
- **Услуга** — точка (запрос «сколько стоит») → выстроенная структура предложения;
  сама подача доказывает уровень.
- **Ниша** — голос клиента: боль как «шум точек» → решение как собранная форма
  (цитаты болей — типографический приём).
- **Кейс** — иммерсивная история: точка (задача) → готовая работа; здесь смелее всего.
- **Статья** — редакционная типографика: буквица, засечки, ритм колонки.
- **О студии** — личность без фото: монограмма из точки, принципы как композиция.
- **Контакты / 404** — тоже сцена: точка как приглашение / потерянная точка.

## Бюджет (потолок, не цель)
Perf ≥ 85 mobile · LCP < 2.5 s · CLS < 0.1 · A11y ≥ 95. Конфликт «эффект vs
метрика» решается не молча в пользу метрики — выносится с ценой в миллисекундах.
Декоративный текст: `aria-hidden` + исходная прозрачность; контраст поднимаем
только у смыслового текста.
