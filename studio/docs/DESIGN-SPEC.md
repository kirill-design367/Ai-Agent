# AUREA — DESIGN-SPEC (эталонный спек, v4)

> Финальная формула. Отменяет предыдущие «догадки о дорогом».
> Три донора, три слоя, одна рука:
> **СКЕЛЕТ = OBYS · ИНТЕРАКТИВ = CUBERTO · ФИЗИКА = LOCOMOTIVE.**
> Разобрано покадрово через ffmpeg (13 записей, 1920×1080). Ниже — что
> измерено и какими значениями перенесено в реализацию.

## 0. Что показал разбор эталонов (факты с кадров)

**OBYS (obys.agency) — скелет и характер**
- База страниц work/about — **инверсно-светлая** (тёплый почти-белый, чёрный
  текст). Тёмное используется точечно, как акцентный экран.
- Дисплейный шрифт — **нейтральный мощный гротеск** (Helvetica-грани), не
  вычурный. «Дорого» рождается **масштабом + строгой сеткой**, не начертанием.
- Микро-навигация растянута по краям верхней строки: слева лого `OBYS®`,
  по центру `Work · About`, справа локальное время + `Contact`.
- Работы поданы как **индекс-список** строками; при наведении в центр
  выезжает превью-миниатюра. Тэги + категории мелким моно.
- Строгая колоночная сетка, тонкие линии на всю ширину, номера секций.
- Точечный резкий акцент (жёлтый на проекте GRIDS) — иначе строгий Ч/Б.

**CUBERTO (cuberto.com) — интерактив**
- Верхняя навигация: лого слева, `Services · Projects · Company · Blog ·
  Contacts` справа. Полноэкранного оверлея на этих записях нет — меню плоское;
  оверлей-паттерн переносим как наш выбор для мобайла/бургера.
- **Кнопки-пилюли и чипы**: контур по умолчанию → на hover тёмная заливка
  **расходится радиально от точки курсора**, текст инвертируется в белый.
  Скорость ≈ 0.45–0.55 s, ease-out. Возврат — обратная реверс-заливка.
- **Магнитность** на ключевых CTA (кнопка тянется к курсору ~6–10 px).
- Контакт: крупный заголовок «Hey! Tell us all the things», **чипы интереса**
  (Site from scratch / UX-UI / Product / Motion / Branding) и **чипы бюджета**
  (10-20k … >100k). Поля формы — только нижняя линия, лейбл-плейсхолдер.
- Финальный CTA — крупный типографический («Have an idea? / TELL·US»),
  пилюля с живым hover. Подвал тёмный, соц-иконки в кружках.
- Курсор-блоб ОТКЛОНЁН владельцем → системный курсор, вся выразительность
  уходит в реакцию самих элементов.

**LOCOMOTIVE (locomotive.ca) — физика**
- Work — тоже **инверсно-светлая** страница-индекс: 3 колонки строкой
  `Название · Категория · Город`, при hover выезжает превью-миниатюра у строки.
- Верх: `Locomotive®` слева, `Work, Agency, Careers, Store` центр,
  `Let's talk` справа; активный пункт подчёркнут.
- Интро секции: слева микро-лейбл `:: All projects ⁴⁵` + счётчик, справа
  абзац-манифест.
- Кейс: миниатюра + тэги (`Digital, E-commerce`) + описание +
  `Case Study →` / `Visit website ↗`; крупное медиа кейса с параллаксом.
- Подвал-карта: Menu / Social / External колонками.
- Скролл — инерционный (Lenis), мягкий разгон, длинное затухание.
- Появление блоков — сдвиг снизу + маска, stagger внутри секции,
  вторичные элементы с запаздыванием.

**Вывод-конвергенция:** OBYS и Locomotive — один язык (светлый редакционный
индекс + hover-превью + краевые микро-лейблы + 3-колоночные мета-строки +
тонкие линии). Все три донора по сути **монохромны**. Значит: убираем
изобретённый янтарь, идём в тёплый Ч/Б с ритмом тёмное↔светлое.

## 1. СЛОЙ OBYS — типографика и композиция

### 1.1 Шрифт (дисплей заменён)
- **Дисплей = Unbounded** (variable 200–900, полная кириллица, OFL, self-host).
  Мощный геометрический гротеск; дозируем — Black для героя/CTA, Medium/Semibold
  для заголовков секций, тугой трекинг. Кириллица проверена живым рендером.
- **Текст = Onest** (спокойный гротеск, оставлен для body/lead).
- **Микро-лейблы = Martian Mono** (uppercase, 10–11px, трекинг ~0.14em).

### 1.2 Масштаб (по пропорции OBYS к вьюпорту)
- Hero H1: дисплейный кегль `clamp(2.6rem, 1rem + 9.5vw, 9rem)` — заголовок
  занимает большую часть первого экрана, во всю ширину.
- Заголовки секций H2: `clamp(2rem, 1rem + 5vw, 4.4rem)`.
- Тип-шкала √φ сохранена для промежуточных ступеней.

### 1.3 Сетка и композиция
- 12-колоночная сетка (`--col: 1fr` ×12, gutter `clamp(12px,1.4vw,28px)`),
  контейнер `min(94vw, 1560px)`.
- Асимметрия по φ: заголовок из 1-й колонки, обрыв ~8-й; подпись в 10–12-й.
- Краевые микро-лейблы, номера секций `01 / 02`, линии-разделители на всю
  ширину контейнера. Активная пустота (большие поля).

### 1.4 Ритм тёмное↔светлое (инверсия как событие)
- База брендовая — тёплый почти-чёрный `#0b0a09`.
- Инверсные секции — тёплый почти-белый `#f3efe6`, тёмный текст. Классы
  `.sec--dark` / `.sec--light`. Смена фона по скроллу — плавная (событие ритма).
- Порядок на главной: dark(hero) → light(манифест/услуги) → dark(кейсы-театр)
  → light(доверие) → dark(CTA). 2–3 смены, не мельтешение.

## 2. СЛОЙ CUBERTO — интерактив (тайминги перенесены)

- **Пилюли/чипы `.btn-fill` / `.chip`**: контур → hover радиальная заливка
  чернил от курсора (`--mx/--my` из JS), текст инверт. `0.5s cubic-bezier(0.65,0,0.35,1)`.
- **Магнит `.mag`** на CTA: сдвиг к курсору ≤10px, возврат spring-ease.
- **Оверлей-меню** (бургер, а также десктоп-«Меню»): полноэкранное, пункты
  дисплейные, вход stagger 60ms снизу-из-маски, длительность панели 0.6s.
- **Форма-контакт**: поля-подчёркивания, чипы интереса + бюджета (инверт на выбор).
- **Финальный CTA** каждой страницы — крупный типографический блок + магнит-пилюля.
- **Page transition**: тёплый занавес (translateY, 0.6s), жест «точки» —
  только здесь (одно место на сайте). Курсор системный.

## 3. СЛОЙ LOCOMOTIVE — физика (Lenis + reveal)

- **Lenis**: `lerp 0.09`, `duration 1.15`, wheelMultiplier 1.0 — мягкий разгон,
  длинное затухание.
- **Reveal**: элементы входят `translateY(28px)+opacity` из маски, внутри
  секции stagger 70ms, вторичные +120ms. Один язык на весь сайт (IO-driven,
  без GSAP на критическом пути; reduce-motion → всё видно сразу).
- **Индекс-работ** = список строк (название·категория·город) + hover-превью,
  которое выезжает у строки (масштаб медиа внутри рамки).
- **Кейс-шаблон**: крупные медиа-слоты с плейсхолдерами (под IMAGE-BRIEF),
  параллакс媒 внутри рамки, hover-scale.
- **Один pin/scrub** на главной — секция кейсов-театра.

## 4. Рамки (неприкосновенно)
- Контент, тексты, SEO (мета/JSON-LD/URL/перелинковка), деплой — не трогаем.
- Первый экран: навигация не наезжает на заголовок (воздух сверху `--header-h`+).
- Логотип — вордмарк AUREA, читаются все 5 букв; guard-тест.
- Кириллица каждого шрифта проверена до применения.
- Перф: LCP < 2.5s mobile, CLS < 0.1, Perf ≥ 85. Конфликт эффект/метрика —
  с ценой в мс.
- 390px — основной трафик: та же формула в мобильном масштабе, hover→touch-state.
- A11y ≥ 95, reduce-motion = без движения.
