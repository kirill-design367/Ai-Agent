# AUREA — DESIGN-DNA

> Единственный источник дизайн-истины. Ведётся по факту принятых решений.
> Прежняя «библия» и дизайн-доки удалены. Основа: промпт «тихая роскошь» +
> 6 референсов (design-refs, Awwwards).

## Закон
- **Два цвета:** чёрный `#0b0b0b` (ink) и белый `#ffffff` (paper). Серые — их
  смеси (вторичный `#6a6a6a`, третичный `#9b9b9b`). Никаких цветных акцентов,
  золота, цветных градиентов. Изображения кейсов — `grayscale(1)`.
- **База страниц — светлая** (белый фон, чёрный текст). Тёмные секции —
  инверсия (`.invert`) как ритмический акцент.
- **Тихая роскошь:** минимализм, воздух главный материал, дороговизна в деталях,
  сдержанные микровзаимодействия. Halo-эффект: каждый hero = «спокойная
  уверенность». Одна цель на секцию.

## Шрифты (self-host woff2, кириллица проверена живым рендером)
- **Дисплей = Druk Cyr** (`--font-display`): Super 900 (гиганты hero/«РАБОТЫ»/
  манифест), Heavy 800, Medium 500 (+ италики). Полная кириллица + ₽.
- **Текст/UI/облако имён = Dexa Round** (`--font-body`): Regular 400, Medium 500,
  SemiBold 600, Black 900 (+ италики). Округлый гуманист.
- Прошлые шрифты (Unbounded, Onest, Playfair, Martian, Oswald, Anton) — удалены
  из проекта (public + package.json).
- Замечание: референсы Artworld/Akoya используют антикву для облака имён;
  антиквы с кириллицей в комплекте нет → облако имён исполнено **Dexa Round**
  (regular + italic микс) — редакционная элегантность в рамках 2 шрифтов.

## Блоки (по референсам)
- **Hero (KOTA/Sidekick):** белый фон, гигантский Druk лесенкой (`--fs-hero`,
  ~11.5vw), строки со сдвигом (`.hx-l1/2/3`); узкая верхняя панель
  (`mix-blend-difference` → авто-контраст); лид справа внизу с `<b>`-выделениями;
  ч/б арт-слот справа (плейсхолдер). Пробелы гарантированы: каждое слово — блок.
- **Кейсы главной (Akoya):** вырезанные `clip-path`-фигуры (трапеции), наклон,
  разброс; grayscale-медиа; имя (италик) / тип по сторонам; контролируемый дрейф
  по скроллу (`data-drift`, разные скорости), hover — выпрямление + масштаб.
- **Хаб кейсов (Artworld):** сверху облако имён проектов (Dexa, микс regular/
  italic, тэги (Л)(М)(В)) — ссылки с hover; снизу гигант «РАБОТЫ» край-в-край.
- **Меню:** полноэкранный оверлей, только слова, гигантские пункты, stagger,
  hover (курсив), внизу мелко контакты. Одинаково десктоп/мобайл.
- **О студии (Lees):** постер — гигантские буквы + мелкий текст-манифест вокруг,
  нумерованные пункты.
- **Блог «чёрная дыра»:** дисторшн перехода (canvas-снимок + шейдер), только
  десктоп, только блог. Мобайл — обычный мягкий переход.

## Движение
- Lenis инерционный скролл. Easing `cubic-bezier(0.16,1,0.3,1)`, длительности
  0.28/0.5/0.8s. Кнопки — заливка чернил снизу; ссылки — вычерчиваемое
  подчёркивание; поля — линия снизу, фокус. reduce-motion — всё статично.

## Рамки
Контент, тексты, SEO (мета/JSON-LD/URL/перелинковка), деплой — не трогаются.
Перф-потолок: LCP<2.5s mobile, CLS<0.1, Perf≥85, A11y≥95.
