# AUREA — Карта интенсивности (§1.3)

> Принцип: **тишина между акцентами и есть роскошь.** На каждой странице — считанные
> «моменты-манифесты» (motion/эффект), остальное — спокойное чтение, где работают
> композиция, типографика и воздух. Непрерывный motion запрещён.
>
> Обозначения: ● момент-манифест · ◐ лёгкий акцент (микро-motion при входе) ·
> ○ спокойная зона (без motion, только композиция/типографика/воздух).

## Бюджет моментов по типам
| Тип страницы | Моментов (●) макс | Логика |
|---|---|---|
| Главная | **3** | манифест студии — можно ярче всего |
| Кейс | **2–3** | витрина работ, здесь смелее (но не непрерывно) |
| Услуга | **1–2** | продаёт подача, не аттракцион |
| Ниша | **1** | голос клиента; один сильный типографический удар |
| О студии | **1** | монограмма/принципы; сдержанно |
| Статья | **~0** | почти чистое чтение, редакционная типографика |
| Контакты | **1** | приглашение (курсор/форма как деталь) |
| 404 | **1** | «точка потерялась» — единственный момент |

---

## Главная (до 3 ●)
- ● **Прелоадер → hero** (только 1-й визит): точка прорастает в композицию.
- ● **Hero-заголовок**: кинетика «сборки из точки», один раз, при входе.
- ● **Кейсы (витрина)**: один срежиссированный scroll-момент (глубина/параллакс).
- ◐ Процесс, цены — лёгкий вход блоков.
- ○ «Что получаете», FAQ, отзывы, финал-CTA — спокойные зоны: типографика + воздух.

## Кейс (2–3 ●)
- ● **Открытие**: обложка/заголовок вырастает из точки-задачи.
- ● **Ключевой разворот работы**: иммерсивный scroll (mockup/параллакс слоёв).
- ◐ Метрики — тихий счётчик при входе (одноразовый).
- ○ Текст истории, галерея — спокойное чтение.

## Услуга (1–2 ●)
- ● **Hero-предложение**: точка (запрос) → выстроенная структура.
- ◐ Состав/цена — лёгкий вход.
- ○ Всё остальное (шаги, FAQ, перелинковка) — спокойно; работает сетка по φ.

## Ниша (1 ●)
- ● **Голос клиента**: боль как «шум» → собирается в одну ясную фразу-решение
  (типографический удар, не аттракцион).
- ○ Решение, калькулятор-описание, FAQ, кейс-ссылка — спокойные зоны.

## О студии (1 ●)
- ● **Монограмма/имя**: проявление из точки.
- ○ Принципы, факты об ИП — композиция и воздух, без motion.

## Статья (~0 ●)
- ○ Полностью спокойное чтение: буквица, засечки, ритм колонки, крупные интертитры.
- ◐ Единственный допустимый микро-акцент — тонкий индикатор прогресса чтения.

## Контакты (1 ●)
- ● **Форма/курсор** как живая деталь (фокус-состояния, отправка).
- ○ Прямые каналы, реквизиты, правовые ссылки — спокойно.

## 404 (1 ●)
- ● **«Точка потерялась»** — единственный момент, затем спокойное предложение вернуться.
