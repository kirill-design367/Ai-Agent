# СЕМЕЙСТВО D. КУРСОР, HOVER, МИКРОВЗАИМОДЕЙСТВИЯ

Уровень тактильности — «хочется потрогать» (Гл. I, 3.1). На мобайле hover/курсор → тап-отклики ⚠️РФ.

### 49. Magnetic Buttons (магнитные кнопки) ★ключевая
Элемент притягивается к курсору при приближении, мягко возвращается.
- **Где:** главные CTA, ключевые ссылки, лого.
- **Нельзя:** на множестве элементов (хаос); мобайл (нет курсора).
- **Психоэффект:** «живой, отзывчивый, физический объект» — сильнейший премиум-микроэффект.
- **Дёшево → дорого:** дёшево — резкий большой сдвиг; дорого — притяжение ≤15px, мягкая пружина, плавный возврат.
- **Реализация:** offset = (cursor−center)×0.2–0.35, `gsap` spring/elastic-out возврат.

### 50. Liquid Cursor (жидкий курсор) ★ключевая
Кастомный курсор с инерцией/текучестью, тянется за мышью.
- **Где:** айдентика, креативный премиум.
- **Нельзя:** функциональные/контентные сайты (мешает); мобайл.
- **Психоэффект:** уникальность бренда, «жидкая материя следует за тобой».
- **Дёшево → дорого:** дёшево — большой лагающий кружок; дорого — тонкая форма с лёгкой инерцией (lerp), мягко догоняющая, не перекрывающая контент.

### 51. Cursor Morph / Context Cursor (контекстный курсор) ★
Курсор меняется над элементами (точка→кольцо→«смотреть»→«тащить»).
- **Где:** портфолио («смотреть кейс»), галереи, интерактив.
- **Нельзя:** мобайл; слишком много состояний.
- **Психоэффект:** курсор «понимает» контекст, ведёт пользователя.
- **Дёшево → дорого:** дёшево — резкая подмена картинки; дорого — плавный morph формы/размера/текста.

### 52. Cursor Trail (след курсора)
Частицы/линия за курсором.
- **Где:** очень дозированно, креатив-бренд.
- **Нельзя:** премиум-сдержанность; ⚠️РФ (нагрузка).
- **Психоэффект:** игривость, «магия».
- **Дёшево → дорого:** дёшево — радужный след (2010-е); дорого — едва заметный монохромный шлейф, быстро тающий.

### 53. Hover Lift (подъём при наведении) ★
Элемент слегка приподнимается + усиление тени.
- **Где:** интерактивные элементы, кнопки, кликабельные блоки.
- **Нельзя:** «только смена цвета» вместо отклика (запрет, Гл. II, 12.3).
- **Психоэффект:** «элемент откликается, приближается ко мне».
- **Дёшево → дорого:** дёшево — резкий jump; дорого — y:−4–8px + тень растёт по высоте (Гл. IV, 12), 200ms expo, плавный откат.

### 54. Hover Image Reveal (образ при наведении на текст) ★
Наведение на пункт списка показывает связанное изображение (часто у курсора).
- **Где:** editorial-списки проектов/услуг (вместо карточек!).
- **Нельзя:** мобайл (нужна альтернатива).
- **Психоэффект:** «оживание» текстового списка, элегантность, отказ от карточек.
- **Дёшево → дорого:** дёшево — резкое появление картинки; дорого — плавный fade+mask+лёгкий параллакс образа за курсором.

### 55. Underline Draw (прорисовка подчёркивания) ★
Подчёркивание ссылки прорисовывается при hover.
- **Где:** все текстовые ссылки, навигация.
- **Нельзя:** —
- **Психоэффект:** аккуратность, отклик, «диалог».
- **Дёшево → дорого:** дёшево — мгновенное появление; дорого — рост left→right, expo, лёгкий, обратимый.

### 56. Button Fill Sweep (заливка кнопки) ★
Фон кнопки «затекает» при hover (снизу/со стороны курсора).
- **Где:** primary/secondary кнопки.
- **Нельзя:** просто смена `background-color`.
- **Психоэффект:** материальность, «жидкое наполнение».
- **Дёш157;во → дорого:** дорого — clip/scale заливки с направлением, expo, текст инвертируется плавно.

### 57. Light Sweep / Glare (блик по поверхности) ★ключевая
Световой блик проходит по элементу при hover/периодически.
- **Где:** карточки-исключения, мокапы, металл-текст, стекло.
- **Нельзя:** на всём; ярко.
- **Психоэффект:** «полированная дорогая поверхность ловит свет».
- **Дёшево → дорого:** дёшево — белая полоса пролетает грубо; дорого — мягкий узкий градиент-блик, медленный, тонкий.

### 58. Glass Reflection (отражение в стекле) ★ключевая
Блик/отражение на стеклянной поверхности реагирует на курсор/скролл.
- **Где:** стеклянные панели, навигация, мокапы экранов.
- **Нельзя:** ⚠️РФ (тяжело на мобайле); поверх текста.
- **Психоэффект:** достоверность материала, «настоящее стекло».
- **Дёшево → дорого:** дорого — динамический градиент-отражение, следящий за движением.

### 59. Tilt / 3D Hover (наклон при наведении) ★ключевая
Элемент наклоняется в 3D к курсору.
- **Где:** мокапы, ключевые медиа, акцентные блоки.
- **Нельзя:** на всём (укачивает); сильный угол; мобайл; ⚠️РФ.
- **Психоэффект:** «физический объект в руке», глубина.
- **Дёшево → дорого:** дёшево — сильный резкий tilt (vanilla-tilt дефолт); дорого — ≤8deg, с глубиной слоёв внутри (parallax внутри карточки), мягкий возврат.

### 60. Hover Scale Media (зум медиа при наведении) ★
Изображение увеличивается в своей рамке при hover.
- **Где:** галереи, превью, портфолио-исключения.
- **Нельзя:** сильный зум (≤1.05–1.1).
- **Психоэффект:** «приближение, интерес».
- **Дёшево → дорого:** дёшево — резкий scale с дёрганьем; дорого — плавный 1.05, overflow-hidden рамка, expo.

### 61. Magnetic Text / Hover Distortion
Текст слегка деформируется/смещается у курсора.
- **Где:** креативные заголовки, лого.
- **Нельзя:** читаемый функциональный текст; мобайл.
- **Психоэффект:** «текст реагирует на присутствие».
- **Дёшево → дорого:** дорого — тонкая волна, не сильное искажение.

### 62. Press / Tap Feedback (отклик нажатия) ★
Лёгкое «вдавливание» при клике/тапе (scale 0.97).
- **Где:** ВСЕ кликабельные элементы (включая мобайл!).
- **Нельзя:** отсутствовать (мёртвый интерфейс).
- **Психоэффект:** тактильное подтверждение, «кнопка настоящая».
- **Дёшево → дорого:** дорого — мгновенный, упругий, едва заметный.

### 63. Toggle / Switch Morph (переключатели)
Плавная анимация переключений.
- **Где:** настройки, фильтры, тема.
- **Нельзя:** резкая подмена состояния.
- **Психоэффект:** «механизм работает гладко».
- **Дёшево → дорого:** дорого — morph с инерцией, не мгновенный скачок.

### 64. Form Field Focus Glow (подсветка поля) ★
Мягкое появление подсветки/границы при фокусе (Гл. IV, 17).
- **Где:** все поля форм (финал!).
- **Нельзя:** резкий синий outline по умолчанию.
- **Психоэффект:** «поле приняло меня», внимание.
- **Дёшево → дорого:** дорого — плавная подсветка акцентом 200ms, не резкий бордер.

### 65. Success Micro-Animation (микро-успех) ★
Анимация подтверждения (галочка morph, отправка формы).
- **Где:** отправка формы, копирование, завершение действия.
- **Нельзя:** унылый alert.
- **Психоэффект:** вознаграждение, завершённость, радость.
- **Дёшево → дорого:** дёшево — текст «Отправлено»; дорого — morph-галочка, мягкий свет, достойный финальный момент (Гл. III, 11.8).

### 66. Hover Sound (звук при наведении/клике)
Тончайший звуковой отклик.
- **Где:** очень дозированно, премиум-айдентика (по согласованию).
- **Нельзя:** по умолчанию вкл; навязчиво; раздражающе.
- **Психоэффект:** мультисенсорная роскошь.
- **Дёшево → дорого:** дорого — выкл по умолчанию, тончайший, дорогой звук.

# СЕМЕЙСТВО E. ИЗОБРАЖЕНИЯ И МЕДИА

### 67. Image Zoom on Scroll (зум по скроллу) ★
Изображение медленно увеличивается/уменьшается при скролле.
- **Где:** full-bleed кадры, hero-медиа, портфолио.
- **Нельзя:** сильный зум; ⚠️РФ.
- **Психоэффект:** «живой кадр», кинематограф, глубина времени.
- **Дёшево → дорого:** дорого — тонкий (1.0–1.1) scrub-зум, плавный.

### 68. Image Reveal Mask (раскрытие образа маской) ★
Изображение проявляется из-под маски (clip-path).
- **Где:** появление медиа в сцене, editorial.
- **Нельзя:** банальным wipe везде.
- **Психоэффект:** «проявление», ценность образа.
- **Дёшево → дорого:** дёшево — fade картинки; дорого — clip-маска снизу/формой, expo, иногда с лёгким зумом внутри.

### 69. Image Slice (нарезка) ★ключевая
Изображение собирается/распадается из полос/сегментов.
- **Где:** акцентное появление кейса, кульминация.
- **Нельзя:** часто; ⚠️РФ.
- **Психоэффект:** «жалюзи раскрываются», техно-элегантность.
- **Дёшево → дорого:** дёшево — грубые прыгающие полосы; дорого — точный stagger сегментов, expo, тонкие зазоры.

### 70. Duotone / Color Grade on Scroll
Цветокоррекция изображения меняется по скроллу/hover (ч-б→цвет).
- **Где:** портфолио, галереи, hover-оживление.
- **Нельзя:** кислотные фильтры.
- **Психоэффект:** «оживание», фокус, кинематографический грейд.
- **Дёшево → дорого:** дорого — тонкий грейд/дуотон, плавный.

### 71. Image Parallax Inside Frame (параллакс внутри рамки) ★
Изображение движется внутри своей рамки при скролле.
- **Где:** галереи, editorial-медиа.
- **Нельзя:** сильный сдвиг.
- **Психоэффект:** глубина, «окно в мир».
- **Дёшево → дорого:** дорого — тонкий, overflow-hidden, плавный.

### 72. Clip Crop Expand (раскрытие кадрирования)
Кадр изображения расширяется от детали к полному.
- **Где:** драматичное раскрытие, переход в кейс.
- **Нельзя:** часто.
- **Психоэффект:** «отъезд камеры», раскрытие контекста.
- **Дёшево → дорого:** дорого — scrub, с глубиной.

### 73. Hover Distort / RGB Shift (искажение/хром.аберрация)
Лёгкое искажение или RGB-сдвиг изображения при hover.
- **Где:** креатив-бренд, портфолио-hover.
- **Нельзя:** строгий люкс; сильно; мобайл; ⚠️РФ (шейдер).
- **Психоэффект:** «глитч-энергия», современность.
- **Дёшево → дорого:** дёшево — сильный некрасивый глитч; дорого — едва заметный, дорогой, краткий.

### 74. Video Scrub (видео по скроллу) ★
Кадр видео привязан к позиции скролла.
- **Где:** продуктовые раскрытия, флагман-кейс.
- **Нельзя:** ⚠️РФ (тяжело, трафик); без оптимизации.
- **Психоэффект:** прямое управление «фильмом», вовлечение.
- **Дёшево → дорого:** дорого — оптимизированное видео/секвенция, плавный scrub, фоллбэк на мобайле.

### 75. Ken Burns (медленный дрейф фото)
Медленное панорамирование+зум статичного фото.
- **Где:** атмосферный фон, hero.
- **Нельзя:** быстро.
- **Психоэффект:** «живой кадр», кинематограф из статики.
- **Дёшево → дорого:** дорого — очень медленно, едва заметно.

### 76. Image Sequence Animation (покадровая секвенция)
Серия кадров проигрывается по скроллу (псевдо-3D вращение объекта).
- **Где:** продукт-витрина, флагман.
- **Нельзя:** ⚠️РФ (вес кадров); без preload.
- **Психоэффект:** «3D-объект в руках», премиум-продукт.
- **Дёшево → дорого:** дорого — оптимизированные кадры, плавно, фоллбэк.

### 77. Masonry / Gallery Stagger
Появление галереи каскадом.
- **Где:** галереи изображений.
- **Нельзя:** одинаковый fade всех разом.
- **Психоэффект:** «волна», ритм.
- **Дёшево → дорого:** дорого — stagger по позиции, разные задержки.

### 78. Before/After Reveal (до/после) ★
Интерактивное сравнение изображений (для «Редизайн»!).
- **Где:** кейс редизайна (контент клиента!), трансформации.
- **Нельзя:** —
- **Психоэффект:** наглядное доказательство ценности, «вот разница».
- **Дёшево → дорого:** дорого — плавный drag/scrub, тонкий разделитель, тактильность.

### 79. Floating Media (парящие медиа)
Образы медленно «парят» (ambient).
- **Где:** hero, атмосфера.
- **Нельзя:** много; быстро.
- **Психоэффект:** «живость», воздушность.
- **Дёшево → дорого:** дорого — очень медленный, асинхронный дрейф.

### 80. Image Stack Shuffle (тасование стопки)
Стопка изображений перелистывается/тасуется.
- **Где:** превью работ в hero (тихая витрина, Гл. III, сцена 1).
- **Нельзя:** быстро; навязчиво.
- **Психоэффект:** «галерея живёт», намёк на объём работ.
- **Дёшево → дорого:** дорого — плавная смена с глубиной/тенью.

# СЕМЕЙСТВО F. ГЛУБИНА, 3D, ПЕРСПЕКТИВА

⚠️РФ — почти всё семейство тяжёлое на мобайле, упрощать/отключать.

### 81. Perspective Shift (сдвиг перспективы) ★ключевая
Точка наблюдения сцены меняется при скролле/движении.
- **Где:** кинематографичные сцены, кульминации.
- **Нельзя:** часто; на функциональном контенте; ⚠️РФ.
- **Психоэффект:** «облёт», пространственность, «вау».
- **Дёшево → дорого:** дорого — тонкий, плавный, мотивированный, с глубиной слоёв.

### 82. 3D Card / Object (3D-объект) ★
Реальный 3D-элемент (WebGL/Three).
- **Где:** флагман-кульминация, продукт, hero-акцент.
- **Нельзя:** везде; ⚠️РФ (тяжело); без фоллбэка.
- **Психоэффект:** «настоящий объект», технологичный премиум.
- **Дёшево → дорого:** дёшево — кривая модель с дефолт-освещением; дорого — выверенный свет, материалы, плавность, лёгкость.

### 83. Depth Hover Layers (слои внутри при hover)
Слои элемента двигаются с разной глубиной при наведении.
- **Где:** ключевые карточки-исключения, мокапы.
- **Нельзя:** на всём; мобайл.
- **Психоэффект:** «заглядываю внутрь», глубина.
- **Дёшево → дорого:** дорого — тонкий параллакс слоёв, мягко.

### 84. Z-Translate Scroll (движение по оси Z)
Элементы «приближаются/удаляются» по Z при скролле.
- **Где:** погружение, переходы.
- **Нельзя:** укачивающе; ⚠️РФ.
- **Психоэффект:** полёт сквозь пространство.
- **Дёшево → дорого:** дорого — с расфокусом дальних, плавно.

### 85. Fog / Depth Fade (глубинный туман)
Дальние слои тонут в дымке/расфокусе.
- **Где:** атмосферные сцены, глубина.
- **Нельзя:** мутя контент.
- **Психоэффект:** «атмосферная перспектива», кино.
- **Дёшево → дорого:** дорого — тонкий градиент-дымка, окрашенная светом.

### 86. Orbit / Rotate Object
Объект медленно вращается (ambient или по скроллу).
- **Где:** продукт, лого-3D, флагман.
- **Нельзя:** быстро; везде; ⚠️РФ.
- **Психоэффект:** «рассмотреть со всех сторон».
- **Дёшево → дорого:** дорого — медленно, со светом, инерцией.

### 87. Parallax Mouse (параллакс от мыши)
Слои сдвигаются от движения курсора.
- **Где:** hero, атмосфера (desktop).
- **Нельзя:** сильно (укачивает); мобайл.
- **Психоэффект:** «3D-окно», глубина, отклик на присутствие.
- **Дёшево → дорого:** дорого — тонкий (≤20px), с инерцией lerp.

### 88. Card Flip (переворот)
Элемент переворачивается, открывая обратную сторону.
- **Где:** дозированно, интерактив-раскрытие.
- **Нельзя:** массово (эффект «визиток 2012»).
- **Психоэффект:** «другая сторона», игра.
- **Дёшево → дорого:** дорого — с тенью/глубиной в процессе, expo.

### 89. Isometric Build (изометрическая сборка)
Сцена собирается в изометрии.
- **Где:** объяснение системы/процесса, техно-бренд.
- **Нельзя:** люкс-минимализм.
- **Психоэффект:** «конструкция собирается», ясность системы.
- **Дёшево → дорого:** дорого — выверенный свет, stagger сборки.

### 90. Spotlight / Light Follow (свет за курсором) ★
Источник света/блик следует за курсором по сцене.
- **Где:** тёмные сцены, hero, акценты (Гл. II, 7.4).
- **Нельзя:** ярко; мобайл.
- **Психоэффект:** «я освещаю мир», интимность, фокус.
- **Дёшево → дорого:** дорого — мягкое радиальное свечение, инерция, тонко.

# СЕМЕЙСТВО G. МАТЕРИАЛ, СВЕТ, АТМОСФЕРА

### 91. Noise / Grain Animation (живое зерно) ★ключевая
Анимированный шум поверх сцены (Гл. I, 10.2; Гл. IV, 10.2).
- **Где:** почти везде как фоновая текстура (3–8%).
- **Нельзя:** сильный, заметный; ⚠️РФ (если тяжёлая реализация — статичный grain).
- **Психоэффект:** «аналоговость», глубина, «снято, а не сгенерировано».
- **Дёшево → дорого:** дёшево — крупный заметный шум; дорого — мелкое, едва дышащее зерно, soft-light.

### 92. Gradient Mesh Animation (живой градиент)
Mesh-градиент медленно перетекает.
- **Где:** атмосферный фон, hero (с шумом!).
- **Нельзя:** диско-цвета (Гл. I, 11.2); без шума (стоково); ⚠️РФ.
- **Психоэффект:** «дышащая атмосфера», глубина.
- **Дёшево → дорого:** дёшево — кислотный анимированный градиент; дорого — приглушённые тона, медленно, с зерном.

### 93. Aurora / Glow Drift (свечение)
Мягкие пятна света медленно дрейфуют.
- **Где:** тёмные сцены, премиум-фон.
- **Нельзя:** ярко; быстро.
- **Психоэффект:** «северное сияние», дорогая атмосфера.
- **Дёшево → дорого:** дорого — приглушённое, медленное, с blur и шумом.

### 94. Liquid / Metaball (жидкая материя)
Сливающиеся текучие формы.
- **Где:** органик-бренд, акцент.
- **Нельзя:** строгий люкс; ⚠️РФ (шейдер).
- **Психоэффект:** органика, гипноз, мягкость.
- **Дёшево → дорого:** дорого — плавный шейдер, приглушённый.

### 95. Caustics / Refraction (преломление)
Имитация преломления света (вода, стекло).
- **Где:** премиум-стекло, продукт.
- **Нельзя:** ⚠️РФ; везде.
- **Психоэффект:** достоверность дорогого материала.
- **Дёшево → дорого:** дорого — тонко, реалистично.

### 96. Smoke / Fog Drift (дым)
Медленный дым/туман.
- **Где:** атмосфера, тёмные кинематографичные сцены.
- **Нельзя:** мутя контент; ⚠️РФ.
- **Психоэффект:** мистика, глубина, кино.
- **Дёшево → дорого:** дорого — тонкий, медленный.

### 97. Shimmer / Skeleton Loading (мерцание загрузки)
Анимация плейсхолдера при загрузке контента.
- **Где:** подгрузка данных/изображений.
- **Нельзя:** грубое серое мигание.
- **Психоэффект:** «система работает», снижение тревоги ожидания.
- **Дёшево → дорого:** дорого — мягкий тональный shimmer, в тон сцены.

### 98. Ambient Float / Breathing (дыхание)
Элемент медленно «дышит» (scale/opacity).
- **Где:** hero-превью (Гл. III, сцена 1), акценты, лого.
- **Нельзя:** много; заметно.
- **Психоэффект:** «живой объект», спокойствие.
- **Дёшево → дорого:** дорого — едва заметно, очень медленно (3s+).

### 99. Light Leak / Lens Flare (блик объектива)
Световые блики как от объектива.
- **Где:** дозированно, кинематографичные кадры.
- **Нельзя:** часто (китч); ярко.
- **Психоэффект:** «снято на дорогую оптику».
- **Дёшево → дорого:** дорого — тонко, уместно, не стоковый PNG-flare.

### 100. Texture Reveal (проявление фактуры)
Фактура (бумага/камень) проступает при появлении сцены.
- **Где:** смена атмосферы, материальные сцены.
- **Нельзя:** грубо.
- **Психоэффект:** материальность, тактильность.
- **Дёшево → дорого:** дорого — тонко, согласовано со светом.

# СЕМЕЙСТВО H. SVG И ВЕКТОР

### 101. SVG Draw (прорисовка контура) ★ключевая
Линия/контур рисуется (stroke-dashoffset).
- **Где:** лого, иллюстрации, схемы, разделители, линия-путь (Гл. III, 4), иконки (Гл. IV, 18).
- **Нельзя:** на всём; медленно (скучно).
- **Психоэффект:** «рисуется на глазах», мастерство, внимание.
- **Дёшево → дорого:** дёшево — линейная равномерная отрисовка; дорого — expo, переменная скорость, по scrub.

### 102. SVG Path Morph (морфинг путей) ★
Векторная форма превращается в другую.
- **Где:** интерактивные иконки, переходы форм, лого-анимация.
- **Нельзя:** между несовместимыми формами (грязно).
- **Психоэффект:** «магия превращения», непрерывность.
- **Дёшево → дорого:** дорого — совместимые точки путей, плавно (flubber/GSAP MorphSVG).

### 103. Animated Line Connector
Линии соединяют элементы анимированно (схема процесса).
- **Где:** «как я работаю», схемы.
- **Нельзя:** хаотичная паутина.
- **Психоэффект:** «связь, поток, система».
- **Дёшево → дорого:** дорого — прорисовка по порядку, expo.

### 104. SVG Filter Animation (анимация фильтров)
Анимация SVG-фильтров (turbulence, displacement).
- **Где:** liquid/distort эффекты, текстуры.
- **Нельзя:** ⚠️РФ (тяжело).
- **Психоэффект:** органика, «живая материя».
- **Дёшево → дорого:** дорого — тонко, оптимизированно.

### 105. Blob Morph (морфинг кляксы)
Органичная форма плавно меняет очертания.
- **Где:** фон-акцент, маска медиа.
- **Нельзя:** строгий люкс; быстро.
- **Психоэффект:** мягкость, органика.
- **Дёшево → дорого:** дорого — медленный, плавный, приглушённый.

### 106. Stroke Width Animation
Толщина обводки меняется выразительно.
- **Где:** лого, акцентная графика.
- **Нельзя:** дёргано.
- **Психоэффект:** «каллиграфия», живость линии.
- **Дёшево → дорого:** дорого — плавно, с ритмом.

### 107. Animated Gradient Stroke
Градиент бежит по контуру.
- **Где:** акцентные рамки, лого.
- **Нельзя:** радужно; на всём.
- **Психоэффект:** «энергия по контуру».
- **Дёшево → дорого:** дорого — монохром/тон, тонко.

### 108. SVG Logo Build (сборка лого)
Лого собирается из элементов/прорисовывается.
- **Где:** прелоадер, hero, финал.
- **Нельзя:** долго (на входе).
- **Психоэффект:** «бренд рождается», айдентика.
- **Дёшево → дорого:** дорого — выверенный stagger/draw, expo, коротко.

# СЕМЕЙСТВО I. LAYOUT / GRID MOTION

### 109. Infinite Grid / Marquee (бесконечная лента) ★ключевая
Бесконечный поток элементов (лого-клиентов, тегов, отзывов).
- **Где:** объём соц-доказательства (отзывы-фон, Гл. III, 9), услуги-теги.
- **Нельзя:** с важным уникальным контентом; быстро.
- **Психоэффект:** «их много», непрерывность, объём.
- **Дёшево → дорого:** дёшево — быстрая бегущая строка; дорого — медленно, с краевым fade-маской, hover-пауза.

### 110. FLIP Layout Animation ★
Плавная анимация при изменении раскладки (фильтрация, сортировка).
- **Где:** фильтры портфолио, перестроение галереи.
- **Нельзя:** резкая перерисовка.
- **Психоэффект:** «всё плавно переехало», непрерывность восприятия (Гл. II, 15.3).
- **Дёшево → дорого:** дёшево — мгновенный rearrange; дорого — GSAP Flip, элементы плавно переезжают.

### 111. Staggered Grid Reveal ★
Сетка/галерея появляется каскадом по позиции.
- **Где:** галереи, перечни (не карточки-клоны!).
- **Нельзя:** одновременный fade всех.
- **Психоэффект:** «волна», ритм, режиссура взгляда.
- **Дёшево → дорого:** дорого — stagger по направлению (диагональ/центр), разные задержки.

### 112. Expanding Grid Item ★
Элемент сетки разворачивается в полноэкранный (галерея→кейс).
- **Где:** портфолио-переход в кейс.
- **Нельзя:** резко.
- **Психоэффект:** «погружение в выбранное», непрерывность.
- **Дёшево → дорого:** дорого — shared-element morph (FLIP), не «открылась новая страница».

### 113. Column Drift (дрейф колонок)
Соседние колонки скроллятся с разной скоростью.
- **Где:** editorial-галереи, портфолио.
- **Нельзя:** сильно; ⚠️РФ.
- **Психоэффект:** «живая верстка», глубина.
- **Дёшево → дорого:** дорого — тонкая разница скоростей, плавно.

### 114. Sticky Side Nav Progress
Боковая навигация отслеживает активную секцию.
- **Где:** длинные нарративы, многосценовые сайты.
- **Нельзя:** загромождая.
- **Психоэффект:** ориентация, контроль.
- **Дёшево → дорого:** дорого — тонкая, с плавным активным индикатором.

### 115. Reveal on Resize / Responsive Morph
Раскладка плавно перестраивается при изменении области.
- **Где:** адаптив, интерактив.
- **Нельзя:** скачками.
- **Психоэффект:** «всё живое и гибкое».
- **Дёшево → дорого:** дорого — плавные переходы между состояниями.

### 116. Bento Reveal (бенто-сетка с жизнью)
Бенто-раскладка появляется/реагирует (если бенто вообще уместно — осторожно с трендом).
- **Где:** дозированно, обзор возможностей.
- **Нельзя:** как замена editorial; клоны-ячейки (Гл. IV, 15).
- **Психоэффект:** структурированный обзор.
- **Дёшево → дорого:** дорого — разные размеры/вес, осмысленный stagger, не равные плитки.

# СЕМЕЙСТВО J. ЗАГРУЗКА И ВХОД

### 117. Preloader / Intro (прелоадер) ★
Заставка-загрузка перед входом.
- **Где:** премиум-сайты, первое впечатление, маскировка загрузки тяжёлых ассетов.
- **Нельзя:** долго (>2–3s раздражает); ради «красоты» на лёгком сайте.
- **Психоэффект:** «спектакль начинается», аванс уровня (Гл. I, 4.5).
- **Дёшево → дорого:** дёшево — спиннер/проценты крутятся; дорого — типографика/лого + curtain reveal в hero, короткий, осмысленный.

### 118. Counter Preloader (счётчик загрузки)
Числа 0→100 как прелоадер.
- **Где:** премиум-интро.
- **Нельзя:** медленно; если грузить нечего (фейк).
- **Психоэффект:** «подготовка чего-то стоящего».
- **Дёшево → дорого:** дорого — крупная типографика, ease-out, переход в curtain.

### 119. First Paint Stagger (вход hero) ★
Срежиссированное появление hero (Гл. III, сцена 1).
- **Где:** каждый hero.
- **Нельзя:** всё разом; дефолтный fade.
- **Психоэффект:** первое впечатление, заявка уровня.
- **Дёшево → дорого:** дорого — таймлайн-партитура (заголовок→подзаголовок→CTA→метрики) с нахлёстом (Гл. II, 4.2).

### 120. Lazy Reveal (появление при подгрузке)
Контент элегантно появляется по мере загрузки.
- **Где:** изображения, тяжёлые секции.
- **Нельзя:** резкий «прыжок» лейаута (CLS).
- **Психоэффект:** плавность, отсутствие рывков.
- **Дёшево → дорого:** дорого — зарезервированное место + мягкий reveal, без сдвига layout.

### 121. Scroll-to-Top / Anchor Smooth
Плавный переход к секции/верху.
- **Где:** навигация, якоря.
- **Нельзя:** мгновенный прыжок.
- **Психоэффект:** непрерывность, контроль (Lenis, Гл. II, 6).
- **Дёшево → дорого:** дорого — инерционный, согласованный с Lenis.

### 122. Exit / Outro Animation
Анимация при уходе со страницы/закрытии.
- **Где:** SPA-переходы, закрытие модалок.
- **Нельзя:** задерживая пользователя долго.
- **Психоэффект:** завершённость, «двери закрываются мягко».
- **Дёшево → дорого:** дорого — симметрично входу, expo, короткий.

## РФ-СВОДКА по Motion Library

Для русскоязычного рынка (мобильный трафик, средние устройства, Гл. IV, 20) действует правило **«премиум остаётся, вес снижается»**:
- все техники с пометкой ⚠️РФ — на мобайле упрощать или отключать, держа 60fps;
- WebGL/шейдеры (3D, liquid, caustics, RGB-shift) — обязательно с лёгким фоллбэком;
- видео-scrub/секвенции — оптимизировать вес, фоллбэк на статику;
- тяжёлый параллакс/3D-tilt → лёгкие версии или отключение;
- но базовый премиум-слой (Mask Reveal, stagger, magnetic, hover-lift, тонкий параллакс, light sweep, тактильные отклики, плавные переходы) — **сохраняется везде**, он лёгкий и даёт 80% ощущения дороговизны;
- понятность важнее эффекта (premium clarity): ни одна анимация не должна мешать РФ-пользователю быстро понять «что, сколько, как связаться» (Гл. III).

## СВОДНЫЕ ЗАПРЕТЫ Главы V

1. Запрещён дефолтный fade-up на всём (главная причина одинаковости, Гл. II, 16.1; техника 19).
2. Запрещено применять >1 «сильной» техники (взрыв, морф, 3D, scroll-theatre) в одной зоне.
3. Запрещено жечь кульминационные приёмы (4, 23, 40, 69, 82) вне кульминаций.
4. Запрещены техники, ломающие 60fps на среднем мобайле без фоллбэка (⚠️РФ).
5. Запрещён hover «только цвет» (53, 55, 56) и отсутствие tap-feedback (62).
6. Запрещена анимация, мешающая читать текст (семейство A) или понять оффер (РФ).
7. Запрещён скролл-хайджекинг без оправдания историей (27, 22, 23).
8. Запрещены кислотные градиенты/радужные эффекты (8, 70, 73, 92, 107).
9. Запрещён прелоадер ради красоты на лёгком сайте дольше 2–3s (117).
10. Запрещено игнорировать `prefers-reduced-motion` для любой техники.
11. Запрещён грубый заметный grain/skew/tilt/zoom — премиум всегда тонок.
12. Запрещено выбирать технику «потому что умею» — только «потому что нужно сцене» (Гл. II, 1.2).

## ЧЕК-ЛИСТ выбора техники

1. Какую функцию несёт (направить/объяснить/создать эмоцию)? Если никакую — не применять.
2. Это кульминационный приём? Тогда он только в кульминации, не раньше?
3. Не повторяется ли приём из соседней сцены (Гл. VI)?
4. Тонко ли настроено (параметры премиума, не дефолт)?
5. 60fps на среднем Android? Есть фоллбэк (⚠️РФ)?
6. Не мешает ли читаемости/пониманию оффера (РФ)?
7. Уважён reduced-motion? Есть tap-feedback на мобайле?
8. Тест замедления x4 (Гл. II, 3.4) — красиво в slow-mo?
9. Тест тишины (Гл. I, 5.3) — сцена держится и без этой анимации?
