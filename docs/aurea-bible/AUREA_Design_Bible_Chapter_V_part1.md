# ГЛАВА V. MOTION LIBRARY

*Энциклопедия премиальных анимаций. 120+ техник. Где, когда, почему и чем отличается дорогое от дешёвого.*

---

## 0. Как пользоваться этой главой

Глава II дала **принципы** движения. Эта глава даёт **приёмы**. Каждый приём — инструмент; брать его в руки можно только держа в голове Главу II: движение мотивировано, иерархично, ритмично, бережётся для нужного места (Гл. II, 1–2).

> Знать 120 техник — не значит применять 120 техник. Мастер узнаваем не числом приёмов на сайте, а **точностью выбора одного**. Половина этой главы — про то, где приём **нельзя**.

**Формат записи каждой техники:**
- **Где** — уместное применение.
- **Нельзя** — где запрещено/вредно.
- **Психоэффект** — что происходит в восприятии.
- **Дёшево → дорого** — что отделяет премиальную реализацию от ширпотреба.
- (для ключевых — **Реализация/Параметры**).

**РФ-пометка** ⚠️РФ — техники, требующие осторожности из-за мобильного трафика на средних устройствах (Гл. IV, 20): на мобайле упрощать/отключать, держать 60fps.

# СЕМЕЙСТВО A. ТЕКСТОВЫЕ АНИМАЦИИ

Главное семейство AUREA — типографика это 80% дизайна (Гл. I, 9).

### 1. Mask Reveal по строкам (Line Mask Reveal) ★ключевая
Текст разбит на строки (SplitText), каждая в контейнере с `overflow:hidden`, строки выезжают снизу из-под невидимой маски каскадом.
- **Где:** заголовки сцен, лиды, ключевые абзацы, hero, финал. Эталонный премиум-приём.
- **Нельзя:** на каждом абзаце сайта (утомляет, Гл. II, 10.4); на длинных текстах-простынях.
- **Психоэффект:** текст «проявляется из глубины бумаги» — ощущение материальности и режиссуры; глаз ведётся построчно (Visual Focus).
- **Дёшево → дорого:** дёшево — резкий fade или одновременное появление всех строк; дорого — выезд из маски, expo.out, точный stagger, нахлёст.
- **Реализация:** `SplitText` lines → `y:100%→0`, `ease:'expo.out'`, `duration:0.9`, `stagger:0.08–0.12`, маска через wrapper `overflow:hidden`.

### 2. Word Reveal (по словам) ★ключевая
Каскадное появление слов.
- **Где:** короткие сильные фразы, акцентные заголовки, манифесты.
- **Нельзя:** длинные тексты (рвёт чтение на слова); вместе со строчным reveal в одном блоке.
- **Психоэффект:** ритмичное «накатывание» смысла, акцент на каждом слове.
- **Дёшево → дорого:** дёшево — слова прыгают с bounce; дорого — мягкий подъём+проявление, stagger 40–80ms, лёгкий blur→sharp.

### 3. Letter Reveal / Letter Animation (по буквам)
Анимация отдельных букв.
- **Где:** одно слово-логотип, короткий hero-акцент, кульминация.
- **Нельзя:** предложения и абзацы (нечитаемо, дёшево); часто.
- **Психоэффект:** максимальная экспрессия, «оживание» текста.
- **Дёшево → дорого:** дёшево — каждая буква скачет/крутится по-цирковому; дорого — тонкий каскад с минимальным смещением и проявлением.

### 4. Letter Explosion (взрыв букв) ★ключевая
Буквы разлетаются/собираются.
- **Где:** ТОЛЬКО кульминация (портфолио-флагман, мощный акцент), одна короткая фраза.
- **Нельзя:** где угодно ещё. Самый передозируемый приём интернета.
- **Психоэффект:** драма, энергия, «разрушение/сборка».
- **Дёшево → дорого:** дёшево — буквы хаотично разлетаются с поворотами (баннер казино); дорого — физически достоверный разлёт с инерцией, окрашенный смыслом, дозированный, expo.

### 5. Text Dissolve (растворение)
Буквы/слова теряют непрозрачность вразнобой при уходе.
- **Где:** исчезновение текста при смене (sticky-этапы, смена кейса), мягкий уход.
- **Нельзя:** для появления важного (растворение читается как «уход»).
- **Психоэффект:** элегантное угасание, «развеивание».
- **Дёшево → дорого:** дёшево — резкий общий fade; дорого — пер-символьный stagger-dissolve с лёгким blur и сдвигом.

### 6. Typography Collapse (схлопывание) ★ключевая
Текст сжимается/складывается при скролле (трекинг→0, вес/размер падает, строки слипаются).
- **Где:** драматичный переход между сценами, «сжатие» заголовка в строку-метку.
- **Нельзя:** часто; на читаемом в моменте тексте.
- **Психоэффект:** ощущение «концентрации», перехода энергии в покой.
- **Дёшево → дорого:** дёшево — простой scale-down; дорого — анимация переменных осей шрифта (variable: weight/width) + трекинг + scrub по скроллу.

### 7. Text Expansion (раскрытие)
Обратное 6: текст «раскрывается» (трекинг растёт, вес тяжелеет).
- **Где:** появление манифеста, нарастание акцента.
- **Нельзя:** на служебном тексте.
- **Психоэффект:** «вдох», нарастание значимости.
- **Дёшево → дорого:** дорого — variable-оси + воздух, не просто scale.

### 8. Gradient Text Sweep (свет по тексту)
Световой блик/градиент проходит по тексту (металл, Гл. I, 10).
- **Где:** логотип, одно акцентное слово, hero-заголовок.
- **Нельзя:** на всём тексте; ярким радужным градиентом.
- **Психоэффект:** ощущение полированного материала, «дорогого металла».
- **Дёшево → дорого:** дёшево — анимированный радужный фон текста; дорого — тонкий монохромный блик, медленный, редкий.

### 9. Text Blur-In (фокусировка)
Текст из размытия в резкость.
- **Где:** мягкое кинематографичное появление, «наведение фокуса».
- **Нельзя:** часто (blur тяжёл ⚠️РФ); на мелком тексте.
- **Психоэффект:** «камера наводит фокус», глубина.
- **Дёшево → дорого:** дёшево — сильный blur+fade резко; дорого — лёгкий blur(8px→0)+подъём, expo, ⚠️РФ на мобайле упростить.

### 10. Scramble / Decode (раскодирование)
Символы «перебираются» и складываются в слово.
- **Где:** технологичный бренд, числа/метрики, одно слово.
- **Нельзя:** премиум-антиква/люкс (читается «гиково»); длинный текст.
- **Психоэффект:** «расшифровка», технологичность, движение к ясности.
- **Дёшево → дорого:** дёшево — долгий хаотичный перебор; дорого — быстрый, короткий, моноширинный, точный.

### 11. Counter / Number Roll (счётчик метрик)
Числа набегают (7 лет, 100+ проектов).
- **Где:** метрики доверия (hero-лента), статистика.
- **Нельзя:** на ценах (манипулятивно), без триггера в зоне видимости.
- **Психоэффект:** «накопление», динамика достижения.
- **Дёшево → дорого:** дёшево — линейный быстрый счёт; дорого — ease-out замедление к финалу, моноширинные цифры (без дёрганья ширины), запуск по ScrollTrigger.

### 12. Highlight Sweep (прорисовка выделения)
Подчёркивание/маркер «прорисовывается» под словом.
- **Где:** акцент на ключевом слове в тексте, hover ссылок.
- **Нельзя:** на многих словах сразу.
- **Психоэффект:** «вот это важно», ведение к смыслу.
- **Дёшево → дорого:** дёшево — мгновенное жёлтое выделение; дорого — тонкая линия, прорисовка left→right, expo.

### 13. Vertical Text Marquee (вертикальная лента слов)
Медленный вертикальный/горизонтальный поток слов-тегов.
- **Где:** фон-атмосфера, список услуг/тегов, заполнение «мёртвой» зоны движением.
- **Нельзя:** с важным текстом; быстро (раздражает).
- **Психоэффект:** «живость», ambient-движение, объём.
- **Дёшево → дорого:** дёшево — быстрая бегущая строка (новостной тикер); дорого — очень медленная, с краевым fade-маской, ambient.

### 14. Kinetic Typography (кинетическая типографика)
Текст синхронизирован с ритмом, двигается выразительно.
- **Где:** заставка, манифест-сцена, бренд с характером.
- **Нельзя:** на функциональном контенте; строгий люкс.
- **Психоэффект:** энергия, ритм, «текст танцует».
- **Дёшево → дорого:** дёшево — случайные прыжки; дорого — выверенный ритм, музыкальная пауза, дисциплина.

### 15. Outline → Fill Text (контур заполняется)
Текст-контур заливается цветом.
- **Где:** крупный hero-акцент, заголовок секции.
- **Нельзя:** мелкий текст; часто.
- **Психоэффект:** «оживание», нарастание присутствия.
- **Дёшево → дорого:** дёшево — резкая заливка; дорого — плавная по scrub-скроллу, направленная.

### 16. Sticky Word Pinning (закреплённое слово)
Ключевое слово остаётся, вокруг меняется текст.
- **Где:** манифест «Я делаю ___ : быстро / красиво / честно».
- **Нельзя:** длинно.
- **Психоэффект:** акцент на постоянстве главного при смене деталей.
- **Дёшево → дорого:** дорого — мягкие morph-смены сменного слова, не резкая подмена.

### 17. Justified Reflow (переверстка по ширине)
Текст переустраивается по ширине при скролле/ресайзе выразительно.
- **Где:** editorial-сцены, крупные заголовки.
- **Нельзя:** часто (тяжело, может дёргать ⚠️РФ).
- **Психоэффект:** «живой набор», редакционность.
- **Дёшево → дорого:** дорого — плавно, без скачков лейаута.

### 18. Char Stagger Hover (буквы реагируют на hover)
При наведении буквы слова реагируют по очереди (подъём/вес).
- **Где:** навигация, ключевые ссылки, лого.
- **Нельзя:** на длинных строках; на мобайле (нет hover).
- **Психоэффект:** тактильность, «текст живой под пальцем».
- **Дёшево → дорого:** дёшево — все буквы прыгают разом; дорого — волна по буквам от позиции курсора, мягкая.

# СЕМЕЙСТВО B. SCROLL-DRIVEN ТЕХНИКИ

Скролл = камера и монтаж (Гл. II, 5).

### 19. Scroll Reveal / Entrance (появление по скроллу)
Базовое появление элемента при входе в зону.
- **Где:** почти везде как база.
- **Нельзя:** одинаковый fade-up на ВСЁМ (главная причина одинаковости сайтов, Гл. II, 16.1).
- **Психоэффект:** «контент рождается по мере внимания».
- **Дёшево → дорого:** дёшево — дефолтный AOS fade-up 0.3s ease на всём; дорого — разные направления/кривые/тайминги под каждую сцену, осознанный stagger.

### 20. Parallax (параллакс) ★ключевая
Слои движутся с разной скоростью при скролле.
- **Где:** создание глубины в hero, портфолио-кадрах, фоне.
- **Нельзя:** на всём; сильный сдвиг; ⚠️РФ тяжёлый параллакс на мобайле.
- **Психоэффект:** иллюзия 3D-глубины, «камера движется сквозь пространство».
- **Дёшево → дорого:** дёшево — сильный очевидный сдвиг, «смотрите параллакс», дёрганый; дорого — тонкий (5–20%), плавный (scrub+Lenis), незаметный как приём, ощутимый как качество.
- **Реализация:** разные `y`-скорости слоёв, `scrub:1`, transform-only.

### 21. Depth Layers (слои глубины) ★ключевая
Многослойная сцена: фон/средний/передний план с параллаксом + расфокусом.
- **Где:** кинематографичные сцены, hero, кульминации.
- **Нельзя:** простые информационные сцены (избыточно).
- **Психоэффект:** настоящая пространственность, погружение.
- **Дёшево → дорого:** дёшево — две картинки внахлёст; дорого — 3–5 слоёв с разной скоростью, blur дальних, свет между слоями.

### 22. Sticky Storytelling / Pinning (закрепление) ★ключевая
Секция «прилипает», скролл проигрывает сцену внутри (Гл. II, 11).
- **Где:** пошаговые процессы (этапы), флагман-кейс, трансформации.
- **Нельзя:** часто (1–2 на сайт); надолго; на негативе; без заметного прогресса.
- **Психоэффект:** «перелистывание слайдов истории», управление временем.
- **Дёшево → дорого:** дёшево — длинный пиннинг с малым изменением (ощущение застревания); дорого — каждый «оборот» = заметный прогресс, внятная длина, индикатор.

### 23. Scroll Theatre (скролл-театр)
Целый акт как закреплённая сцена со светом/декорациями/актёрами под управлением скролла.
- **Где:** флагманская кульминация, презентация одного большого кейса.
- **Нельзя:** более одного на сайт; ⚠️РФ тяжёлый на мобайле.
- **Психоэффект:** полное погружение, «спектакль».
- **Дёшево → дорого:** дорого — срежиссированный свет и камера, не просто «много анимаций на пине».

### 24. Scroll Scrub Animation (скраб) ★ключевая
Анимация жёстко привязана к позиции скролла (Гл. II, 5.2).
- **Где:** параллакс, морфинг, прорисовка, любая сцена «под управлением скролла».
- **Нельзя:** для микровзаимодействий (они по своему таймингу).
- **Психоэффект:** прямое управление камерой, «я двигаю сцену».
- **Дёшево → дорого:** дёшево — резкий scrub за каждым зубцом колеса; дорого — `scrub:1` сглаживание + Lenis, текучесть.

### 25. Scroll Velocity Skew (наклон от скорости)
Элементы слегка наклоняются/растягиваются по скорости скролла.
- **Где:** галереи, портфолио-проезд, динамичные сцены.
- **Нельзя:** строгий люкс; сильный скос (тошнотворно); ⚠️РФ.
- **Психоэффект:** «инерция материи», динамика, живость.
- **Дёшево → дорого:** дёшево — сильный резиновый перекос; дорого — едва заметный skew (≤4deg), затухающий.

### 26. Scroll Snap Scenes (привязка к сценам)
Скролл «защёлкивается» на сценах.
- **Где:** портфолио-кинокадры, презентация-слайды.
- **Нельзя:** на длинных текстовых сценах (мешает чтению).
- **Психоэффект:** «каждый кадр — законченное высказывание».
- **Дёшево → дорого:** дёшево — жёсткий резкий snap; дорого — мягкий магнитный доводчик.

### 27. Horizontal Scroll Section (горизонтальная секция)
Вертикальный скролл → горизонтальное движение (Гл. II, 5.4).
- **Где:** галерея, таймлайн, смена ритма-оси (удивление, Гл. VI).
- **Нельзя:** для текста; без подсказки направления; ⚠️РФ на мобайле осторожно.
- **Психоэффект:** смена оси = смена ритма, «проезд камеры».
- **Дёшево → дорого:** дёшево — дезориентирующий резкий захват; дорого — плавный, с индикатором, ощущением контроля.

### 28. Pinned Progress Indicator (прогресс пиннинга)
Индикатор хода внутри закреплённой сцены.
- **Где:** при любом pinning/scroll theatre.
- **Нельзя:** —
- **Психоэффект:** снимает тревогу «сколько ещё», даёт контроль.
- **Дёшево → дорого:** дорого — тонкий, элегантный, не отвлекающий.

### 29. Scroll-Triggered Counter
См. 11, запуск по входу в зону.

### 30. Reveal on Scroll Direction (реакция на направление)
Разное поведение при скролле вверх/вниз (напр. навигация прячется/появляется).
- **Где:** хедер, плавающие элементы.
- **Нельзя:** дёргано, при каждом микродвижении.
- **Психоэффект:** интерфейс «понимает намерение».
- **Дёшево → дорого:** дорого — с порогом и задержкой, плавно.

### 31. Scroll Progress Bar (полоса прогресса страницы)
- **Где:** длинные нарративные сайты, статьи.
- **Нельзя:** на коротких; кричаще.
- **Психоэффект:** ориентация, «я продвигаюсь».
- **Дёшево → дорого:** дорого — тончайшая, в тон, ненавязчивая.

### 32. Parallax Text over Media (текст-параллакс над медиа)
Текст и фон-медиа движутся с разной скоростью.
- **Где:** full-bleed кадры с заголовком, editorial.
- **Нельзя:** ухудшая читаемость текста.
- **Психоэффект:** глубина, «текст плывёт над миром».
- **Дёшево → дорого:** дорого — тонко, с сохранением контраста текста.

### 33. Sticky Stacking Cards (накопление слоёв)
Сцены/панели «наезжают» друг на друга при скролле стопкой.
- **Где:** последовательность сильных высказываний, кейсы.
- **Нельзя:** для разнородного контента; ⚠️РФ много слоёв.
- **Психоэффект:** «накопление», переход глубиной.
- **Дёшево → дорого:** дёшево — простой z-stack без проработки; дорого — со светом/тенью/масштабом уходящего слоя.

### 34. Scroll-Linked Color Shift (смена палитры по скроллу)
Фон/тема плавно меняется по ходу сайта.
- **Где:** связь сцен, смена настроения (свет↔тьма, Гл. III).
- **Нельзя:** резко, дёргано.
- **Психоэффект:** «путешествие через атмосферы», цельность.
- **Дёшево → дорого:** дорого — плавная интерполяция фона/текста/акцента, согласованная с драматургией.

### 35. Scroll Zoom (Dolly) (наезд камеры) ★
Элемент/сцена увеличивается или уменьшается при скролле (Гл. II, 5.4).
- **Где:** вход в кейс (zoom-in), раскрытие контекста (zoom-out), переходы.
- **Нельзя:** на всём; резко; ⚠️РФ тяжёлый зум.
- **Психоэффект:** «камера приближается/отдаляется», кинематограф.
- **Дёшево → дорого:** дёшево — резкий CSS scale; дорого — плавный scrub-зум с глубиной и расфокусом краёв.

# СЕМЕЙСТВО C. ПЕРЕХОДЫ МЕЖДУ СЦЕНАМИ

Переход = монтажная склейка (Гл. II, 8). Самое слабое место чужих сайтов.

### 36. Fade / Dissolve Transition (растворение)
- **Где:** базовый мягкий переход, спокойные стыки.
- **Нельзя:** там, где нужен «удар»/контраст.
- **Психоэффект:** мягкость, непрерывность, премиум-спокойствие.
- **Дёшево → дорого:** дёшево — резкий opacity; дорого — dissolve с лёгким сдвигом/масштабом, окрашенный светом.

### 37. Mask Reveal Transition (маска) ★ключевая
Новая сцена прорезается сквозь старую через раскрывающуюся форму.
- **Где:** кинематографичные переходы, вход в кейс.
- **Нельзя:** на каждом стыке (приберечь).
- **Психоэффект:** «прорыв в новый мир», театральность.
- **Дёшево → дорого:** дёшево — простой wipe полосой; дорого — органичная форма-маска (clip-path/SVG), expo, со светом по краю.

### 38. Curtain Reveal (занавес) ★ключевая
Сцена открывается как занавес (сверху/снизу/в стороны).
- **Где:** открытие сайта (прелоадер→hero), вход в важную сцену.
- **Нельзя:** часто; медленно (раздражает на входе).
- **Психоэффект:** «спектакль начинается», торжественность.
- **Дёшево → дорого:** дёшево — два div'а разъезжаются линейно; дорого — expo, с тенью/светом кромки, иногда с лого по центру.

### 39. Section Morph (морфинг секций) ★ключевая
Элемент одной сцены превращается в элемент следующей.
- **Где:** бесшовные переходы, связь сцен (Гл. III, переходы).
- **Нельзя:** между несвязанными сценами; ⚠️РФ сложный морф.
- **Психоэффект:** «единое непрерывное пространство», магия.
- **Дёшево → дорого:** дёшево — две отдельные анимации, выглядит как совпадение; дорого — настоящий FLIP/shared-element morph, форма реально перетекает.
- **Реализация:** FLIP-техника (GSAP Flip) или shared layout.

### 40. Zoom-Through Transition (пролёт сквозь) ★
Камера влетает в элемент → новая сцена.
- **Где:** вход в кейс из портфолио, драматичный переход.
- **Нельзя:** часто; ⚠️РФ тяжёлый.
- **Психоэффект:** «погружение внутрь», сильнейшая кинематографичность.
- **Дёшево → дорого:** дорого — с расфокусом, ускорением-замедлением, светом в «тоннеле».

### 41. Push / Slide Transition (выталкивание)
Новая сцена выталкивает старую.
- **Где:** «главы», слайдовые презентации.
- **Нельзя:** на длинном контенте.
- **Психоэффект:** «перелистывание», прогресс.
- **Дёшево → дорого:** дорого — с инерцией и тенью кромки, не плоский слайд.

### 42. Hard Cut (жёсткая склейка)
Резкая смена.
- **Где:** контраст, «удар», смена ритма (Гл. II, 8.1).
- **Нельзя:** часто (рублено).
- **Психоэффект:** шок, акцент, энергия.
- **Дёшево → дорого:** дорого — на сильном смысловом контрасте, осознанно.

### 43. Light Wipe Transition (световой переход)
Сцена «засвечивается» и проявляется новой.
- **Где:** смена свет↔тьма, торжественные переходы.
- **Нельзя:** часто; ярко (слепит).
- **Психоэффект:** «вспышка перемены», театр света.
- **Дёшево → дорого:** дорого — мягкая волна света, не белая вспышка.

### 44. Liquid / Blob Transition (жидкий переход)
Текучая форма перекрывает и раскрывает сцену.
- **Где:** органичный бренд, игривый премиум.
- **Нельзя:** строгий люкс; ⚠️РФ (WebGL тяжёл).
- **Психоэффект:** органика, мягкость, «живая материя».
- **Дёшево → дорого:** дёшево — дёрганый SVG-морф; дорого — плавный шейдер/тонкий клип, физичный.

### 45. Page Transition (SPA-переход между страницами) ★
Бесшовный переход между «страницами» без перезагрузки-ощущения.
- **Где:** переход в полный кейс, разделы.
- **Нельзя:** разрывать ощущение единого пространства резким релоадом.
- **Психоэффект:** «приложение, а не сайт», цельность, премиум.
- **Дёшево → дорого:** дёшево — белая вспышка перезагрузки; дорого — shared-element morph, оверлей-маска, сохранённый контекст (Barba/View Transitions API).

### 46. Scale Transition (масштабное) ★
Сцена/элемент масштабируется при смене.
- **Где:** акцентные переходы, фокус.
- **Нельзя:** резко.
- **Психоэффект:** «приближение важного».
- **Дёшево → дорого:** дорого — с правильной точкой трансформации и кривой.

### 47. Clip-Path Reveal (геометрическое раскрытие)
Сцена раскрывается по геометрической форме (круг, диагональ, многоугольник).
- **Где:** editorial-переходы, акценты.
- **Нельзя:** банальной формой везде.
- **Психоэффект:** редакционная точность, «вырез».
- **Дёшево → дорого:** дорого — выразительная форма + expo, со светом кромки.

### 48. Cross-Scene Element Persistence (сквозной элемент)
Один элемент (лого, линия, образ) живёт через несколько сцен, связывая их.
- **Где:** нарративные сайты, мотив-связка (линия-путь, Гл. III, 4).
- **Нельзя:** —
- **Психоэффект:** непрерывность истории, «нить Ариадны».
- **Дёшево → дорого:** дорого — плавная передача элемента между секциями (shared layout), не «появился заново».
